# 🍕 Fast React Pizza Co.

An elegant, fully-functional pizza ordering web application built with **React**, **React Router (v6)**, and **Redux Toolkit**, styled beautifully using **Tailwind CSS**.

## ✨ Features

- **Dynamic Menu:** Fetches live pizza data directly from a remote API.
- **Advanced State Management:** Utilizes Redux Toolkit to manage the shopping cart, user profile, and order priority.
- **Smooth Form Actions:** Implements React Router `actions` and `loaders` for seamless data fetching and form submission without traditional page refreshes.
- **In-place Updates:** Uses React Router's `useFetcher` to update order priority (Make Priority) instantly in the background without navigation.
- **Geolocation API:** Automatically fetches the user's GPS coordinates (Latitude/Longitude) to pinpoint their delivery address.
- **Responsive Design:** Fully optimized for mobile, tablet, and desktop screens using Tailwind CSS.

## 🛠️ Tech Stack

- **Frontend:** React.js (Vite)
- **Routing:** React Router v6 (Loaders, Actions, Fetchers)
- **State Management:** Redux Toolkit (@reduxjs/toolkit)
- **Styling:** Tailwind CSS
